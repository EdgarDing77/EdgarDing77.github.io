# 微服务监控



## Spring Boot Admin

> 实现微服务监控

服务名称：sc-admin：8150

服务位置：djj-monitor -> sc-admin

配置说明：

```yml
server:
  port: 8150

spring:
  application:
    name: sc-admin
  security:
    user:
      name: 'admin'
      password: 'admin'
  cloud:
    nacos:
      discovery:
        metadata:
          user.name: ${spring.security.user.name}
          user.password: ${spring.security.user.password}
```
